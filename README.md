# Art Rotation Co. (ARC) 🎨🔄

Welcome to the official repository for **Art Rotation Co.**, a unique "Rent-to-Own" hybrid model for contemporary art. 

Built as a high-performance, static site hosted on GitHub Pages, this platform manages the catalog for Mike Andrew Stamper's original artworks, including watercolors, acrylics, and abstract wood sculptures.

## 🚀 The ARC Model
Unlike traditional galleries, ARC allows collectors to live with the art before committing to full ownership.

* **Subscription:** 4% of Fair Market Value (FMV) per month.
* **Ownership Credit:** 75% of all rental fees are credited toward the final purchase price.
* **Rotation:** Members can swap pieces at any time for a flat **Rotation Fee ($75–$125)**, covering "White Glove" transport and installation.

## 🛠 Tech Stack
* **Frontend:** HTML5 / CSS3 / Jekyll (GitHub Pages)
* **Payments:** Stripe Checkout & Subscription Links
* **Asset Management:** Optimized high-res imagery formatted for Saatchi Art compatibility.

## 📈 Integration
This site acts as the "Service Layer" for my **Saatchi Art Portfolio**. While Saatchi handles direct sales, ARC provides the flexibility of rotation and long-term acquisition for private collectors.
